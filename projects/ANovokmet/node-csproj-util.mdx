<Overview>

Ever wanted to write C# code generators? Do you find Visual Studio Extensions too slow to develop. You don't like T4 templates? Using this package you can add files and classes to your existing Visual Studio projects. 
This allows you to create code generators, or any other program that manipulates files in a project. All you need is Node.js.

</Overview>
