<Overview>

[FINADICT](https://finadict.tech/) is a simple and easy to use WebApp for Stocks / Cryptocurreny / Foreign Exchange prices prediction. It's fully built in **Python**, uses the **Yahoo Finance API** to fetch raw data and then **Facebook Prophet** model to fit the historical data and predict future prices of an entity / currency. The frontend is built using **Streamlit**.
Currently, prediction is supported for 1 day interval with maximum of 1 day prediction. Other features include (but are not limited to):
  - Historical data fetch interval ranging from **5 mins** to **1 month**.
  - Viewing Raw historical data in table.
  - Option to download each and every piece of information.
  - Different types of plots for the data.
  - Option to run locally or on a server using docker.

</Overview>
