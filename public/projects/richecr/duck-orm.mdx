<Overview>

The Duck-ORM package is an asynchronous ORM for Python, with support for Postgres and SQLite. ORM is built with:

- databases

Requirements: Python 3.8+

Duck-ORM is still under development.

For this project you need to use Poetry to manage the dependencies to install you can follow the tutorial [here](https://python-poetry.org/docs/#installation)

To run in development, start with the following command:

`$ poetry shell`

Install project dependencies:

`$ poetry install`

To run the tests, just run:

`$ poetry run pytest .`

</Overview>
